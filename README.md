# ETHブラックジャックの入れ方
本アプリはReact環境で動作し、metamaskが導入済みである必要があります。

## Reactの環境を用意する。
Node.jsを使用し、React環境を準備する
`create-react-app`

## ファイルを書き換える
`git clone https://github.com/e205718/BlackJackv1.git`
public と　srcファイルを置き換える。

## moduleのインストール
`npm install react-router-dom`
`npm install web3`

## 起動
`npm start`
